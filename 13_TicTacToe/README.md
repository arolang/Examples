# 13_TicTacToe

## Prompt

> Create an interactive terminal TicTacToe game. The human plays X, the computer plays O and never loses (strategy: win > block > center > corner > edge). Use arrow keys to move a highlighted cursor, Enter to place. Show the board with box-drawing characters. Support restart with 'r' and quit with 'q'/'Esc'. Hide the terminal cursor.

## Run

```bash
aro run ./ExamplesExtra/13_TicTacToe
```

## Keys

- `↑` / `↓` / `←` / `→` — Move cursor
- `Enter` — Place X
- `r` — Restart game
- `q` / `Esc` — Quit
