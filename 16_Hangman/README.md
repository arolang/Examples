# 16_Hangman

## Prompt

> A Hangman word game: the computer picks a word, shows blanks, the player guesses letters one at a time with the keyboard. Draw the gallows progressively with ASCII art.

## Run

```bash
aro run ./Examples/16_Hangman
```

## Keys

- `a`-`z` — Guess a letter
- `q` / `Esc` — Quit
