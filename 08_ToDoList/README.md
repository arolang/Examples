# 8_ToDoList

## Prompt

> Add, view, and remove tasks in a simple interactive cli application.

## Run

```bash
aro run ./ExamplesExtra/8_ToDoList
```

## Keys

- `a` — Add a new task (type text, Enter to confirm, Esc to cancel)
- `↑` / `↓` — Navigate tasks
- `d` — Delete selected task
- `q` / `Esc` — Quit
